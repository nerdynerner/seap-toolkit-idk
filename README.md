# SEAP 2.0 Campus Quest...Loading

An interactive web prototype designed to help University of Alberta staff and students explore key themes from the Student Experience Action Plan 2.0


## Built With

* HTML
* CSS
* JavaScript
* GitHub Pages

## Features

* Interactive campus-map style layout
* Thematic “quest” areas based on SEAP 2.0 priorities
* Undergraduate and graduate student experience toggle
* Completion certificate interaction
